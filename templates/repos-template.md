## Recommended Entry Format

- Repo Name. [[link]](...)
  - Category: control / RL / MPC / simulator / retargeting / dataset / toolkit / viewer
  - Robot Type: humanoid / quadruped / general
  - Simulator: MuJoCo / Isaac / Raisim / PyBullet / other
  - Deploy: sim / hardware / both / browser / data
  - Summary: concise long-term description
  - Notes: optional context, strengths, limitations, or related tools

---

## Maintenance Rules
- Use collapsible `<details>` sections for major function groups.
- Do not include daily-only `Importance` scores here.
- Prefer repositories worth long-term reference.
- Keep entries GitHub-readable and short enough to skim.

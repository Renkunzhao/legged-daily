## Recommended Structure

Use collapsible topic sections in this style:

```md
<details>

  - Notes: ...

</details>
```

## Recommended Entry Format

- Paper Title. [[link]](...)
  - Source: arXiv / ICRA / IROS / RSS / RA-L / T-RO / CoRL / CVPR / NeurIPS / AAAI / IJRR / Science Robotics
  - Date: YYYY-MM-DD
  - Authors: ...
  - Topics: ...
  - Summary: concise long-term description
  - Notes: optional retrieval notes, relation to nearby work, key ideas, or follow-up comments

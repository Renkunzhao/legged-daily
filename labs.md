# Labs

> Curated source-network list for labs, professors, and adjacent researchers relevant to legged robotics.

This draft is an initial split from the current `README.md` researcher section.
The goal is not only to preserve known labs, but also to maintain a discovery network through public sources such as homepages, Google Scholar, arXiv, GitHub, X, LinkedIn, YouTube, and project pages.

---

## How to Use This File

Use this file as a **source network**, not just a static lab directory.
Each entry should help future discovery by making it easy to revisit:
- who to track
- which public sources to watch
- what topics they are associated with
- how they connect to nearby labs, coauthors, or repositories

Suggested growth paths:
- coauthors on newly found papers
- project pages
- repository maintainers
- advisor / lab affiliations
- recurring authors in priority venues
